# SoyUI

* Module-based partial UI
  * pairs nicely with: Bartender4, WeakAuras, Quartz

## Modules

* SoyFilter
  * Filter chat based on matched words
* SoyTooltip
  * uses class color for name
  * adds guild
  * moves tooltip
